# Terminal Portfolio

Just testing this out, you can check it out [here](https://kirousek.eu)

## Development

1. Clone this repo

```shell
git clone https://github.com/kirousek.eu
```

2. Install dependencies

```shell
npm install
```

3. Run development server

```shell
npm run dev
```
